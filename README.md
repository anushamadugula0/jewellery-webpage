# 💎 Anu Jewels – Luxury Jewelry Website

Anu Jewels is a modern and responsive luxury jewelry website designed to showcase premium jewelry collections with an elegant user interface and seamless user experience. The project focuses on luxury aesthetics, responsive layouts, and clean design principles using HTML5 and CSS3.

---

## 🌟 Features

- Luxury-themed homepage design
- Sticky navigation bar
- Interactive hero section
- Featured jewelry showcase
- Collections gallery
- Brand story section
- Customer testimonials
- Appointment booking form
- Responsive design for desktop, tablet, and mobile devices

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Responsive Web Design
- Flexbox
- CSS Grid
- UI/UX Design Principles

---

## 📸 Website Sections

### 🏠 Homepage
Elegant hero section with luxury branding and call-to-action buttons.

### 💍 Featured Product
Highlights premium jewelry pieces with product details.

### ✨ Collections
Displays different jewelry collections through a responsive gallery.

### 📖 Our Story
Showcases the brand's heritage, craftsmanship, and values.

### ⭐ Testimonials
Customer reviews and feedback section.

### 📅 Appointment Booking
Allows users to request appointments through a contact form.
